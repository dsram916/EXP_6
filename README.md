## **Experiment 6.1 – Controlled Form in React**

### **Objective**
To design and implement a controlled form in React where all form input values are managed using React state and updated through event handlers.

---

### **Tech Stack Used**

- **React JS** – Frontend library for building user interfaces  
- **JavaScript (ES6)** – Logic and event handling  
- **HTML5** – Form structure and input elements
